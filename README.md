# Forum digest

A Claude Code skill for tracking users and topics across LessWrong, EA Forum, and Alignment Forum, generating digests of recent activity with special attention to belief updates and mind changes.

## Documentation

See [SKILL.md](./SKILL.md) for complete documentation and usage instructions.

## Installation

To use this skill with Claude Code, clone it into your skills directory:

```bash
git clone https://github.com/HartreeWorks/claude-skill--forum-digest.git ~/.claude/skills/forum-digest
```

Then restart Claude Code to load the skill.
